About

Righter is a free Python library for correcting English texts. We aim to write good, tested code. Ready to be embeded in educational and professional applications.

We have researched the market for good essay assessment solutions. While many exist (even free ones like Language Tools and Pyenchant), none quite fitted the bill. We hope to improve upon the ideas used on those solutions and, ultimately, develop a new — and hopefully more powerful — tool for the Pythonist toolbelt.

It is currently developed by @tatiana and @aflag. Its development repository can be found on github.com/ef-ctx/righter. Feel free to drop by and open issues with bugs and suggestions. Pull requests are also very much welcomed.
